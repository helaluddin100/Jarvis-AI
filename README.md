# Jarvis – Personal AI Assistant

তোমার ব্যক্তিগত AI অ্যাসিস্ট্যান্ট। কথা বলবে এবং লোকাল PC কন্ট্রোল করবে।

---

## ভাষা ও টেক স্ট্যাক (Language & Tech Stack)

### ভাষা: **Python**
- AI/LLM, স্পিচ, সিস্টেম কন্ট্রোলের জন্য সবচেয়ে সহজ ও শক্তিশালী
- বিনামূল্যে লাইব্রেরি অনেক
- macOS/Linux/Windows এ চলে

---

## বিনামূল্যে API / সেবা (Free APIs)

### ১. কনভারসেশন (AI চ্যাট)
| সেবা | বর্ণনা | কিভাবে ফ্রি |
|------|--------|-------------|
| **Ollama** | লোকালে মডেল চালাও (Llama, Mistral ইত্যাদি) | ১০০% ফ্রি, কোনো API key নাই |
| **Groq** | ক্লাউডে দ্রুত রেসপন্স | ফ্রি টিয়ার: ৩০ req/min, ১৪,৪০০ req/day, কার্ড লাগে না |
| **OpenAI** | ChatGPT-style | নতুন অ্যাকাউন্টে $৫–১৮ ক্রেডিট (সীমিত সময়) |

**সুপারিশ:** শুরুতে **Ollama** দিয়ে লোকালে চালাও (প্রাইভেসি + সীমা নেই)। দ্রুত ক্লাউড চাইলে **Groq**।

### ২. স্পিচ টু টেক্সট (কথা → টেক্সট)
| সেবা | বর্ণনা | কিভাবে ফ্রি |
|------|--------|-------------|
| **SpeechRecognition + Google** | মাইক থেকে টেক্সট | লাইব্রেরি ফ্রি, Google এ কিছু লিমিট |
| **Whisper (লোকাল)** | OpenAI-র ওপেন সোর্স মডেল | লোকালে চালালে সম্পূর্ণ ফ্রি |
| **Whisper API** | OpenAI-র পেইড API | ফ্রি নয় – এড়িয়ে চলো |

**সুপারিশ:** অফলাইনে চাইলে **Whisper** লোকাল, ইন্টারনেট থাকলে **SpeechRecognition + Google**।

### ৩. টেক্সট টু স্পিচ (টেক্সট → কণ্ঠ)
| সেবা | বর্ণনা | কিভাবে ফ্রি |
|------|--------|-------------|
| **pyttsx3** | অফলাইন TTS | সম্পূর্ণ ফ্রি, কোনো API নাই |
| **gTTS (Google TTS)** | অনলাইন | ফ্রি (rate limit আছে) |

**সুপারিশ:** অফলাইনের জন্য **pyttsx3**।

### ৪. PC কন্ট্রোল
- **Python:** `subprocess`, `os`, `pyautogui`, `keyboard` – সব ফ্রি, কোনো এক্সটার্নাল API নাই।

## প্রোজেক্ট স্ট্রাকচার

```
JarvisAI/
├── README.md           # এই ফাইল
├── requirements.txt    # Python প্যাকেজ
├── config.py           # সেটিংস (Ollama/Groq, স্পিচ ইত্যাদি)
├── main.py             # মূল এন্ট্রি – জারভিস চালু
└── src/
    ├── ai/             # LLM (Ollama/Groq) সংযোগ
    ├── speech/         # স্পিচ টু টেক্সট, টেক্সট টু স্পিচ
    ├── control/        # PC কন্ট্রোল (অ্যাপ খোলা, ভলিউম, কমান্ড)
    └── voice_agent.py  # ভয়েস লুপ: শোনা → AI → জবাব/কন্ট্রোল
```

## সেটআপ (Setup)

```bash
# ১. Virtual environment (optional কিন্তু ভালো)
python3 -m venv venv
source venv/bin/activate   # macOS/Linux
curl -fsSL https://ollama.com/install.sh | sh; 
# ২. প্যাকেজ ইন্সটল
pip install -r requirements.txt

# ৩. Ollama (যদি লোকাল AI ব্যবহার করো)
# https://ollama.ai থেকে ডাউনলোড করে ইন্সটল করো, তারপর:
ollama pull llama3.2

# ৪. ভয়েস সিকিউরিটি (একবার করলেই হবে)
python -m src.voice_enroll
# ৩ বার একই বাক্য বলো, যেমন: "আমি জারভিসের মালিক"

# ৫. জারভিস চালু
python main.py
```

**ভয়েস মোড:** মাইকে বাংলা/ইংরেজিতে বলো, জারভিস কণ্ঠে জবাব দেবে।  
**ভয়েস রিস্ট্রিকশন:** শুধু **তোমার** ভয়েসে কাজ করবে। অন্য কেউ বললে PC তৎক্ষণাৎ স্লিপে চলে যাবে। বন্ধ করতে বলো: **বন্ধ করো** / **quit** / **exit**।

## সংক্ষিপ্ত সুপারিশ

- **ভাষা:** Python  
- **কথা বলার AI:** Ollama (লোকাল, ফ্রি) অথবা Groq (ক্লাউড, ফ্রি টিয়ার)  
- **কথা → টেক্সট:** SpeechRecognition + Google অথবা Whisper লোকাল  
- **টেক্সট → কণ্ঠ:** pyttsx3 (অফলাইন ফ্রি)  
- **PC কন্ট্রোল:** Python দিয়ে subprocess / pyautogui – সব ফ্রি  

এগুলো দিয়েই তুমি পুরো জারভিস প্রোজেক্ট বিনামূল্যে বানাতে পারবে।
